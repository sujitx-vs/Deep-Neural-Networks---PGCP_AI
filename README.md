# Deep-Neural-Networks---PGCP_AI

##  Daily Deep Neural Networks Practice

Welcome to my repository dedicated to daily coding practice for the **Post Graduate Certificate Program in Artificial Intelligence (PGCP-AI)**. This repo serves as a daily log of my hands-on journey into mastering Deep Neural Networks (DNNs). 

Every day, I implement a new concept using **Google Colab** and push the notebook directly here.

---

## 🚀 How to Use This Repository

Because all practice is done on Google Colab, you **do not** need to install anything locally on your machine.

1. **Open a Notebook:** Click on any "Open in Colab" badge in the table above.
2. **Enable GPU (Crucial for Deep Learning):** Once the notebook opens in Colab, go to the top menu and select:
   > `Runtime` ➡️ `Change runtime type` ➡️ Select **T4 GPU** (or any available hardware accelerator).
3. **Run the Code:** Press `Shift + Enter` to run the cells sequentially.
